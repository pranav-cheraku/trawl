# Trawl Frontend

The Next.js 14 (App Router) frontend for Trawl, a RAG-powered product spec generator.

For prerequisites, environment setup, and how to run the full stack, see the [root README](../README.md).

Quick reference once your environment is configured:

```bash
npm install
npm run dev      # dev server on http://localhost:3000
npm run lint     # ESLint
npx tsc --noEmit # type check
```
